PROJETO: Castle Insper
GRUPO: Lucas Andrade, Rafael Theodoroe e Victor Lucius.
Para rodar o jogo é necessário importar as seguintes bibliotecas dentro do VSCode: “pygame”, “random” e “os”. Na pasta CastleInsper.py execute o programa que dará início ao jogo.
DESCRIÇÃO DO JOGO: O objetivo do jogo é coletar todos os tesouros espalhados pelo mapa. O jogador começa com 10 pontos de vida (HP) e precisará enfrentar diversos inimigos ao longo do caminho. Cada batalha representa um desafio que exige estratégia e atenção, tornando a jornada até o último tesouro uma verdadeira prova de sobrevivência e habilidade.
